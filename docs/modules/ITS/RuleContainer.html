SYNOPSIS

    use ITS;
    use ITS::RuleContainer;
    my $ITS = ITS->new('xml', doc =>'/path/to/doc.xml');
    my ($container) = $ITS->get_containers();
    my $params = $container->params;
    for my $param (my ($name, $val) = each %{$params}){
        print "$name = $val\n";
    }
    for my $rule (@{ $container->rules }){
        print $rule->type . "\n";
    }

DESCRIPTION

This package represents a container for ITS rules. Rule containers have information on the employed ITS version, the query language used in selectors, and parameters available to contained rules. This package is a way to store this information and make it available to the ITS *Rules contained.

METHODS

new

Creates a new RuleContainer object. The first argument should be the original element which this container is representing (its:rules). The others are named arguments:

version

The version of ITS used in the container.

query_language

The query language used by the rules in the container. This defaults to 'xpath', which indicates the use of XPath 2.0 in ITS.

params

A hashref containing all params available to the rules in the container (both those declared in the container and those declared in the container that included this one via xlink:href).

script

The script element containing this RuleContainer as text. This is applicable only to HTML documents.

rules

An arrayref of ITS::DOM::Elements representing the rules inside this container.

element

Returns the original ITS::DOM::Element represented by this container

version

Returns the version of ITS employed by this container.

query_language

Returns the name of the query language employed by the rules in this container.

rules

Returns an arrayref containing ITS::Rule objects for the ITS rules declared in this container.

params

Returns a hashref containing names and values of the parameters available to this container.

script

Returns the script element which holds the return value of element, but as text. Applicable only to HTML documents
