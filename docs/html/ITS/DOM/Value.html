NAME

ITS::DOM::Value - thin wrapper around underlying XML engine value objects

SYNOPSIS

    use ITS::DOM;
    use feature 'say';
    my $dom = ITS::DOM->new(xml => 'path/to/file');
    my @nodes = $dom->get_xpath('"some string"');
    for my $node (@nodes){
        say $node->value;
    }

DESCRIPTION

This module is meant for internal use by the ITS::* modules only. It is a thin wrapper around an XML::LibXML::(Literal|Boolean|Number) objects. There are only two methods (besides new), which are held in common with ITS::DOM::Node.

METHODS

new

Creates a new value object to wrap the input XML::LibXML object.

type

Returns a string representing the type of the node: LIT (for literal, or string), NUM or BOOL.

value

Returns the underlying Perl value of the type represented by this object: a string, a boolean, or a number.

as_xpath

Returns the value of this object as an XPath expression.
