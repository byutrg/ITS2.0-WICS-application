NAME

ITS::WICS::XML2XLIFF::CustomSegmenter - Extract trans-units using custom segmentation (internal use only)

EXPORTS

The following function may be exported:

extract_convert_custom

This extracts groups of trans-unit elements from the input document root using a list of elements to be used for groups and another of those to be used as trans-units. Any elements inside of an element extracted as a trans-unit will be an inline mrk element.

Arguments are the document root, an array ref of the group elements, an array ref of the trans-unit elements, and the match index. Returns (as a list) an array ref of group elements and an array ref of standoff elements.
